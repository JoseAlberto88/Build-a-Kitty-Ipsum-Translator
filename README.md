# Build a Kitty Ipsum Translator

This repository contains the project files for the **Build a Kitty Ipsum Translator** workshop — a 140‑lesson, hands‑on course that teaches intermediate Bash scripting, text processing, and command‑line automation. The project runs entirely inside a Linux environment using **GitHub Codespaces** and is guided by the interactive **CodeRoad** extension.

---

## 🚀 Project Overview

Throughout this workshop, you will:

- Work with real text files (kitty ipsum & doggy ipsum)
- Use tools like `sed`, pipes, redirection, and pattern matching
- Build scripts that transform text automatically
- Learn how commands work under the hood in a Linux shell
- Practice debugging using stdout and stderr logs

This repository serves as the working directory for all lessons, scripts, and outputs generated during the course.

## ▶️ How to Start the Course

1. Create or log into your GitHub account.
2. Open this repository in **GitHub Codespaces**.
3. In VS Code:
   - Open the Command Palette  
   - Run: `CodeRoad: Start`
4. Select **Start New Tutorial** → **URL** tab.
5. Paste the course URL provided by freeCodeCamp.
6. Click **Load**, then **Start**.

Follow the CodeRoad instructions to complete all 140 lessons.
