# Build a Kitty Ipsum Translator

This repository contains the project for the **Build a Kitty Ipsum Translator** course — a 140‑lesson workshop that teaches more complex shell commands and how they work under the hood.

You will work in a virtual Linux machine using **GitHub Codespaces** and follow an interactive tutorial powered by the **CodeRoad** extension.

## Course Overview

- Learn intermediate Bash and shell scripting
- Practice text processing with tools like `sed`, pipes, and redirection
- Work with fun “kitty ipsum” and “doggy ipsum” text files
- Understand how commands, streams, and files interact in a real environment

## Codespaces Setup (Summary)

1. Create or log into your GitHub account.
2. Start a new Codespace for this project.
3. Open the **Command Palette** and run `CodeRoad: Start`.
4. Choose **Start New Tutorial** → **URL** tab.
5. Paste the course URL provided by freeCodeCamp and click **Load**, then **Start**.

If you want to save progress to your freeCodeCamp account, follow the token and secret setup instructions on the course page.
